ecommerce: anteater's cafe

* Introduction *

The main idea of this project is to create an eCommerce website, in which my partner and I decided to sell exotic food online and have components where the users are able to interact with a fully functioning website.

* CSS/Javascript *

This is my first time using JavaScript in a project. The website has JavaScript capabilities of being able to have pictures rotate, check inputs to ensure they are in the correct form and report the current time when the user is on the website.

* Result/Conclusion/Reflection *

This is my first time creating a functioning interactive website with the usage of PHP and I was just so mind-blown and proud of the website that my partner and I collaborated together to create. This project introduced me to the world of web designing
